# Gradient-descent-linear-regression
Implemented Gradient Descent from scratch to train a linear regression model and visualized loss convergence, parameter updates, and model fitting to understand optimization fundamentals.
The objective is to understand how optimization works internally without using ML libraries.

y = mx + b
Parameters m and b are updated iteratively to minimize Mean Squared Error.

What I Implemented:
Linear regression using gradient descent
Manual slope and intercept updates
Mean Squared Error loss calculation
Iterative parameter optimization
Loss tracking across epochs
Model prediction function
Visualization of convergence
